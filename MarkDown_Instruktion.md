### Заголовок третьего уровня

Вывод логов
``` cd
git log
```
Вывод логов в укароченом виде
``` cd
git log ---oneline
```

Перемещение между комитами
```cd
git checkout <индетификационный номер>
```
Возвращение к последним изменениям
```cd
git ma
```
